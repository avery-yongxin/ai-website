# 抖音电商素材工作台

面向抖音电商日常上新场景的运营提效工具。运营可以导入商品、生成图文草稿、编辑结果、收藏模板并复用，把分散的素材和历史方案沉淀成可管理的内容资产。

## 功能

| 页面 | 说明 |
|------|------|
| **工作台首页** | 生产看板，展示今日产能、模板复用率、整体生产流程 |
| **素材生成** | 填写商品信息（名称、类目、品牌、材质、价格等），选择模板，一键生成主图草稿、标题和卖点文案 |
| **结果编辑** | 对生成的草稿进行标题、卖点、风格、标签的微调，实时预览 |
| **模板库** | 将高转化组合保存为模板，下次上新直接套用 |
| **素材库** | 集中管理已生成的商品图、参考图、贴纸素材、文案素材 |
| **运营报告** | 内容效率趋势、热门模板排行、待优化商品提醒 |

## 技术栈

- 纯原生 JavaScript（ES2021+），无框架依赖
- CSS 自定义属性 + Grid Layout 响应式布局
- localStorage 数据持久化
- 单页应用（SPA）路由

## 使用

直接在浏览器打开 `index.html`：

```
# 如果是本地静态服务器
npx serve .
```

1. 进入 **素材生成** 页，点击「填入测试商品」快速体验
2. 点击「生成素材」模拟生成主图草稿和文案
3. 进入 **结果编辑** 调整内容
4. 将满意的方案收藏到 **模板库**
5. 下次上新时选择模板快速复用

无需构建、无需数据库、无需 API 服务。
